# A Step Ahead of Drought: Forecasting Global Water Storage Challenge

Zindi competition: Predict one-month ahead Total Water Storage (TWS) globally.

## Goal
Win the competition (1st place - €1,000)

## Strategy
- Strong feature engineering from Copernicus covariates
- Spatial-temporal modeling
- Bias documentation for report phase
- Innovation in multi-source satellite data fusion

## Data
- Train: 2,154,021 rows
- Test: 280,961 rows
- Features: TWS_t, SPEI_01-12, SOIL_MOISTURE_t
- Target: TWS_t+1

## Evaluation
- Leaderboard: RMSE (50%)
- AI Trustworthiness (30%)
- Innovation & Practicality (20%)

## Deadline
Sep 13, 2026
